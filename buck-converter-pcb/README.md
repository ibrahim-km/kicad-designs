# 48V to 12V Buck Converter PCB

A high-efficiency SMPS buck converter designed in KiCad using the LM2576HVS-12 regulator to step down 48V DC to a regulated 12V DC output.

## Status

Completed

## Tools Used

- KiCad
- LM2576HVS-12
- Power Electronics Design
- PCB Layout Design
